Environment Setup and Execution Guide

Language Version
Python 3.8 or higher

IDE Used
- PyCharm Community Edition

Required Modules
- Ensure you have Python 3.8 or higher installed on your machine.
- This project does not require external modules outside of Python's standard library.

How to Run
1. Open your terminal or command prompt.
2. Navigate to the project directory where the files are located.
3. Ensure the following files are present in the directory:
   - `__init__.py`
   - `__main__.py`
   - `prepostfix.py`
   - `stack_class.py`
   - Your input file (e.g., `input.txt`) containing prefix expressions.
4. Run the program using the command:
python -m lab1 <input_file_path> <output_file_path>
